# Agritech App

**Agritech App** est une application web dédiée à l’agriculture et à la formation agricole. Elle fournit des informations sur la météo locale, les indicateurs de la FAO, ainsi que des formations pratiques pour les agriculteurs et professionnels du secteur agricole.

## Aperçu
Agritech App offre une interface moderne et responsive pour accéder à des informations utiles dans le domaine agricole :  

- Indicateurs de la FAO pour le Niger.  
- Prévisions météo locales.  
- Formations pratiques pour les agriculteurs.  
- Informations sur les membres de l’équipe et leurs spécialités.  
- Newsletter et contact pour rester informé des dernières nouvelles agricoles.

---

## Fonctionnalités
- Navigation responsive avec menu latéral et barre de navigation.  
- Sections interactives avec des cartes superposées et animations au survol.  
- Présentation des formations et des indicateurs avec détails supplémentaires.  
- Pied de page complet avec informations de contact, réseaux sociaux et newsletter.  
- Compatible sur tous les écrans grâce à Bootstrap.

---

## Technologies utilisées
- HTML5  
- CSS3  
- [Bootstrap 5](https://getbootstrap.com/)  
- [Bootstrap Icons](https://icons.getbootstrap.com/)  
- JavaScript (via Bootstrap pour interactions de composants)
